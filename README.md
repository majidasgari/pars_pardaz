# Pars Pardaz

# Features
This library is purely written in Kotlin language and supports these features:
1. Standard corpus making
1. Text cleaning
1. Sentence tokenizing
1. Word tokenizing
1. Lemmatizing
1. POS tagging
1. Dependency Parsing
